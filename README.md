# 🚢 Titanic Data Analysis Project

## 📊 Overview
This project performs data cleaning and exploratory data analysis (EDA) on the Titanic dataset using Python.

---

## 🧹 Data Cleaning
- Handled missing values:
  - Age → median imputation
  - Embarked → mode imputation
  - Cabin → dropped due to high missing values
- Checked and confirmed no duplicate records

---

## 📈 Exploratory Data Analysis
- Females had higher survival rate than males
- First class passengers had highest survival rate
- Overall survival rate was around 38%

---

## 📊 Visualizations
- Survival Count
- Gender vs Survival Rate
- Passenger Class vs Survival Rate

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 🎯 Conclusion
This analysis shows survival was highly influenced by gender and passenger class.
